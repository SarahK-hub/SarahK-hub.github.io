📝 Gestionnaire de tâches – C++

BTS SIO SLAM | Projet C++ | Application Console

💻 Voir le code

Description du projet

Ce projet consiste à développer une application de gestion de tâches en C++. L'utilisateur peut ajouter, consulter, modifier ou supprimer des tâches depuis un menu interactif en ligne de commande.

Le programme utilise des structures de données simples et des fonctions pour organiser la logique de gestion des tâches.

Fonctionnalités

  • Ajout d'une tâche
  • Affichage de la liste des tâches
  • Marquage d'une tâche comme terminée
  • Suppression d'une tâche
  • Menu interactif en console

Technologies utilisées

  • C++
  • Structures (struct)
  • Tableaux
  • Fonctions
  • Entrées / sorties console

Compétences BTS SIO – SLAM

  • Développer une application en langage compilé
  • Manipuler des structures de données
  • Créer une interface console interactive
  • Structurer un programme en plusieurs fonctions

Ce projet m'a permis de comprendre l'organisation d'une application console en C++ et la gestion d'une logique de données simple.
