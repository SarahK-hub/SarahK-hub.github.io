🎮 Jeu du Morpion – Python

BTS SIO SLAM | Projet Python / Algorithmique | Type : Application Console Interactive

Voir le code sur GitHub

Description du projet

Développement d’un jeu du Morpion en Python en mode console. L’objectif était de concevoir une application interactive intégrant une logique algorithmique complète, la gestion des tours de jeu et la détection automatique des conditions de victoire.

Fonctionnalités

  • Grille dynamique 3x3
  • Alternance automatique des joueurs
  • Validation des entrées utilisateur
  • Détection victoire (lignes, colonnes, diagonales)
  • Gestion du match nul
  • Structure modulaire avec fonctions séparées

Compétences développées

  • Algorithmique avancée
  • Manipulation de listes 2D
  • Gestion des boucles imbriquées
  • Utilisation de la fonction all()
  • Validation et contrôle des données utilisateur
  • Organisation modulaire d’un programme Python

Objectifs pédagogiques (BTS SIO SLAM)

  • Développer une application interactive
  • Structurer un programme en modules
  • Implémenter une logique métier complète
  • Tester et sécuriser les entrées utilisateur

Ce projet démontre ma capacité à concevoir un algorithme complet et structuré, allant de la conception logique à l’implémentation fonctionnelle.
