Description du projet

Ce projet est une machine à sous développée en JavaScript. L’objectif est de manipuler le DOM, gérer des événements utilisateurs et créer un rendu visuel interactif de type casino.

Fonctionnalités

  • Système de mise et de crédits
  • Animation des rouleaux
  • Gestion des gains et pertes
  • Super Jackpot avec animation spéciale
  • Bouton activation / désactivation du son
  • Thèmes graphiques selon le niveau

Technologies utilisées

  • HTML5
  • CSS3 (animations, thèmes)
  • JavaScript (DOM, événements, logique de jeu)
  • Bootstrap

Compétences BTS SIO – SLAM

  • Manipuler le DOM en JavaScript
